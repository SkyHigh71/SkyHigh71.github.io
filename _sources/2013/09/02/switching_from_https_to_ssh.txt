switching from HTTPS to SSH
===========================

Communication protocol for github can either be SSH or HTTPS.
You switch change protocol to SSH so that ssh key is used for authentication and you can avoid password authentication.

URL, which includes protocol, is specified in .git/config file.

.. code-block:: ini

    [remote "origin"]
        url = https://github.com/<USERNAME>/<REPO>.git

So as to change URL, use "git remote set-url" command.

.. code-block:: bash

    $ git remote set-url origin git@github.com:<USERNAME>/<REPO>.git

Url in config file shall be updated as specified in argument.

.. code-block:: ini

    [remote "origin"]
        url = git@github.com:<USERNAME>/<REPO>.git
